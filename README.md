## Install, Run and Contribute

**Install packages**

`npm install`

**Compiles and hot-reloads for development**

`npm run serve`

**Compiles and minifies for production**

`npm run build`

**Run unit tests**

`npm run test:unit`

**Run end-to-end tests**

`npm run test:e2e`

**Lints and fixes files**

`npm run lint`
